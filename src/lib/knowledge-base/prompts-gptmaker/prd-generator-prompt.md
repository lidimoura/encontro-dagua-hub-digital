# Gerador de PRD (Product Requirements Document)

## Papel
Atue como Product Manager Sênior.

## Tarefa
Crie um PRD para [PRODUTO] focando em:

1. **Problema**: Qual dor estamos resolvendo?
2. **Público**: Quem é o usuário ideal?
3. **Valor Central**: Por que isso importa?
4. **Jornada do Usuário**: Passo a passo da experiência
5. **Métricas de Sucesso**: Como medimos o impacto?

## Formato
Use linguagem clara, objetiva e orientada a resultados.
